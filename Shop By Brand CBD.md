## 🛠️ Technical Approach (Playwright POM)
The solution utilizes the **Page Object Model (POM)** for high maintainability:
1. **Dynamic Hrefs**: Targeting brands by unique URL slugs (`/brand/...`) instead of fragile image selectors.
2. **Context Resets**: Navigating back to the Home page between interactions to ensure a clean state for the menu.
3. **Wait Strategies**: Used `waitForSelector` with a 10s custom timeout to handle lazy-loading grids.
